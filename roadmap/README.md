# Roadmap

Project planning and task management for JidoWorkspace.

## Quick Start

1. **Planning a new milestone**: Copy `templates/PLAN_TEMPLATE.md` to `workspace/milestone-1.md` or `projects/[name]/milestone-1.md`
2. **Adding tasks**: Use `templates/TASKS_TEMPLATE.md` for `backlog.md` files  
3. **Brain dumping ideas**: Add to `ideas.md` files or create new ones
4. **New project**: Create `roadmap/projects/[project-name]/` with `backlog.md` and `ideas.md`

## Structure

- `workspace/` - Cross-project planning and ideas
- `projects/` - Per-project planning (mirrors main projects/ folder)  
- `templates/` - Copy-paste ready markdown templates

## Documentation

See [AGENTS.md](AGENTS.md) for complete documentation and workflow details.

## File Naming

- `milestone-N.md` - Milestone-based plans (sequential numbering)
- `backlog.md` - Organized tasks and structured ideas  
- `ideas.md` - Raw brain dumps and inspiration
- `v1.2.md` - Version-based planning (optional)
