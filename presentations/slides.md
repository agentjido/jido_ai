# Jido
## Composable Action Framework

Building intelligent, composable actions for AI-driven workflows

---

# What is Jido?

- **Composable**: Actions that can be chained and combined
- **Validated**: Schema-driven parameter validation
- **AI-Ready**: Native tool definitions for LLM integration
- **Executable**: Sync/async execution with retries and timeouts

---

# Core Components

- **Jido.Action** - Define validated, composable actions
- **Jido.Exec** - Execution engine with error handling
- **Jido.Instruction** - Workflow composition system
- **Jido.Plan** - DAG execution plans with dependencies
- **Jido.Tools.*** - 25+ pre-built tools
