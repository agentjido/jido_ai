    Universal LLM client built as Req plugins.

    Provides multi-provider AI support with cost tracking, token counting, and streaming capabilities.

    Functions

    attach(req, opts \\ [])

    @spec attach(
  Req.Request.t(),
  keyword()
) :: Req.Request.t()

    Attach ReqAI plugins to a Req request.

    Options

    • :plugins - List of plugins to attach. Defaults to [:auth, :token_counter, :cost_tracker]
    • :provider - AI provider (:openai, :anthropic, etc.)
    • :model - Model identifier
    • :api_key - API key or key resolution spec
    • :stream - Enable streaming responses (default: false)

    Examples

    # Basic text generation
req = Req.new()
|> ReqAI.attach(provider: :openai, model: "gpt-4", api_key: "sk-...")

response = Req.post!(req, json: %{
  messages: [%{role: "user", content: "Hello"}]
})

# With streaming
req = Req.new()
|> ReqAI.attach(provider: :openai, model: "gpt-4", stream: true)

# Selective plugins
req = Req.new() 
|> ReqAI.attach(plugins: [:auth, :token_counter], provider: :openai)
