    ReqAI

    View Source

    Universal LLM client built as Req plugins.

    Provides multi-provider AI support with cost tracking, token counting, and streaming capabilities.

    Installation

    If available in Hex, the package can be installed by adding req_ai to your list of dependencies in mix.exs:

    def deps do
  [
    {:req_ai, "~> 0.1.0"}
  ]
end

    Usage

    # Attach AI plugins to a Req request
req = Req.new() |> ReqAI.attach(provider: :openai, model: "gpt-4")

# Make requests with AI-specific features
response = Req.post!(req, json: %{
  messages: [%{role: "user", content: "Hello!"}]
})
