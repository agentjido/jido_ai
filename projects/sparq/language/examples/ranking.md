# Ranking of Existing Examples by Complexity

1. **base.md**
   - High-level summaries of potential agents and their key responsibilities.
   - Serves more as an outline rather than a full, complex script.

2. **dr-suess.md**
   - Contains whimsical dialogue but remains fairly straightforward in structure.
   - Introduces concurrency in a limited way but is mostly focused on playful style.

3. **brand-voice.md**
   - Focuses on social media coordination, includes multiple characters, and some parallel flows.

4. **code-mentor.md**
   - Multi-character code review scenario with branching and parallel flows.

5. **parallel_investigation.md**
   - Demonstrates concurrency across multiple characters investigating a problem together.

6. **culinary-companion.md**
   - More complex meal-planning scenario with parallel tasks and specialized tools.

7. **wellness-guide.md**
   - A comprehensive health coaching system with parallel data gathering and multiple characters.

8. **script-architect.md**
   - A meta-level agent creation system; demonstrates advanced concurrency, tool usage, and script validation flows. Likely the most sophisticated example.
